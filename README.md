# Fitness AI App

React Native (Expo) ile geliştirilmiş yapay zeka destekli bir fitness ve sağlık uygulamasıdır. Kullanıcıya özel antrenman programları oluşturur ve günlük beslenme, su ve kalori takibini sağlar.

---

## Özellikler

- Yapay zeka ile kişiye özel antrenman programı oluşturma  
- Günlük kalori takibi  
- Su tüketimi takibi  
- Beslenme planı oluşturma  
- Yemek ve içecek tarifleri önerme  
- Kullanıcıya özel fitness hedefleri  

---

## Amaç

Bu uygulama, kullanıcıların sağlık ve fitness hedeflerine ulaşmalarını kolaylaştırmak için yapay zeka destekli kişisel bir asistan olarak geliştirilmiştir.

---

## Kullanılan Teknolojiler

- React Native  
- Expo  
- JavaScript / TypeScript  
- AI API (OpenAI vb.)  
- Async Storage / Local Storage
